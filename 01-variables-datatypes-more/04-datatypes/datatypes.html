Data types in javascript

  1. Primitive Data Types
    1. String - Sequence of characters. Must be in quotes of backticks
    2. Number - Integers as well as floating point numbers
    3. Boolean - Logical entity / true or false
    4. Null - Intentional absence of any object value
    5. Undefined - A variable that has not yet been defined / assigned
    6. Symbol - Built-in object whose constructor returns a unique symbol
    7. Bigint - Numbers that are greater than the "Number type can handle"
  2. Reference Data Types or Objects
    • Reference types or "objects" are a non-primitve value and when assigned to a variable, the variable is given a reference to that value.
    • Object literals, arrays and functions are all reference types

Static Typing vs Dynamic Types

  • javascript is a dynamically-typed language. This means, we do not explicitly define the types for our variables. many other languages are statically-types such as C, C++ and Java
